Food Security Risk

Fetch Food Security and Famine Risk

Enter a country name or ISO3 code to fetch FEWS NET and IPC snapshots as separate datasets, including Phase 3/4/5 ADM1 hotspots and trend percentages. FEWS compares the latest projection period with the previous projection period (typically around 3 months when available).

FEWS NET data is loaded from a GitHub Actions snapshot that uses repository secrets server-side, then refreshes the latest snapshot for this country. IPC analysis still loads live from the IPC API.
Data comes from FEWS NET Data Warehouse IPC endpoints. Treat this as rapid situational awareness and verify final decisions against official FEWS NET and IPC publications.
Advisory data is fetched live from official sources with source-specific parsers. Always verify on the official government pages before making travel decisions.
